# BlockBuddies4Life Website

Official website for BlockBuddies4Life gaming content channel.

## About

BlockBuddies4Life creates epic gaming content for kids and teens featuring Minecraft, Roblox, and Fortnite. This website serves as the official web presence and provides Terms of Service and Privacy Policy for TikTok API integration.

## Content

- **Homepage**: Channel introduction and links to social platforms
- **About**: Information about our content creation mission
- **Terms of Service**: Legal terms for content usage and API integration
- **Privacy Policy**: Privacy practices and COPPA compliance
